# AI Document & Audio Processor

> 📋 Универсальный AI-обработчик: **PDF→Excel, аудио→текст, OCR, batch-извлечение полей** в одном инструменте.

## Для бизнеса

**Что это:** Веб-сервис, который автоматизирует рутину обработки документов и аудио через AI. Заменяет ассистента, который вручную перепечатывает счета, расшифровывает встречи и заполняет таблицы.

**Категория спроса:** устойчивый спрос на OCR, анализ документов, транскрибацию (составной из 3 прежних проектов).

**Кому пригодится:**
- 👥 HR — пачка резюме → таблица с навыками
- ⚖️ Юристы — анализ договоров, риски, ключевые условия
- 💰 Бухгалтерия — счета и КП → таблица для 1C
- 🎙 Подкастеры — транскрибация эпизодов с тайм-кодами
- 💼 Менеджеры — протоколы встреч с задачами
- 🏥 Медцентры — выписки → структурированная карта

**Что получают:**
- ✅ Пакетная обработка PDF/DOCX/аудио
- ✅ OCR для сканов
- ✅ Извлечение полей в Excel
- ✅ Транскрибация аудио с тайм-кодами
- ✅ Резюме длинных встреч и документов через GPT
- ✅ Кастомные Pydantic-схемы под клиента

**Цена внедрения:** 5 000 — 50 000 ₽

## Для разработчика

**Стек:**
- Python 3.12 + **FastAPI** + **Streamlit** UI (upload + results display)
- **PyPDF / pypdfium2** — текстовые PDF
- **Tesseract** + `pytesseract` — OCR сканов
- **OpenAI Whisper API** — аудио
- **OpenAI GPT-4o-mini / GigaChat** — извлечение полей
- **`instructor`** + Pydantic — structured output
- `pandas` + `openpyxl` — Excel

**Архитектура:**
```
┌──────────── Upload ────────────┐
│  PDF / DOCX / MP3 / MP4 / WAV  │
└────────────────┬───────────────┘
                 ↓
   ┌─────────────┴─────────────┐
   ↓                            ↓
[Document Pipeline]      [Audio Pipeline]
   ↓                            ↓
[PyPDF / Tesseract]     [Whisper API]
   ↓                            ↓
[Text + structure]      [Transcript + timestamps]
   ↓                            ↓
[LLM + Pydantic schema] [GPT summary + action items]
   ↓                            ↓
   └─────────────┬─────────────┘
                 ↓
   [Excel / CSV / DOCX / SRT / JSON]
```

**Целевые метрики (замерим при реализации):**
- Throughput на пачку 100 документов
- Cost per документ / per минута аудио
- Точность извлечения полей на эталонах
- Точность транскрипции на русском

**Готовые шаблоны:**

**Документы:**
1. Договор → стороны, сумма, сроки, штрафы, риски
2. Резюме → ФИО, опыт, навыки, образование, ожидания
3. Счёт/КП → реквизиты, позиции, суммы, сроки
4. Отчёт → выводы, цифры, рекомендации
5. Кастомный → пользователь задаёт Pydantic-схему через UI

**Аудио:**
1. Подкаст → транскрипт + SRT-субтитры
2. Встреча → протокол + список задач
3. Лекция → конспект с разделами
4. Интервью → выделение цитат

**Что демонстрирует:**
- Multi-format pipeline (документы + аудио)
- OCR integration
- Structured extraction через `instructor`
- CLI + FastAPI + Streamlit UI
- Pydantic schemas as configuration

## Правовые дисклеймеры

- Документы с ПДн обрабатываются по 152-ФЗ — клиент несёт ответственность
- Whisper API: данные не используются для обучения (per OpenAI ToS)
- Для медицинских / юридических данных рекомендуется локальный Whisper
- При обработке аудио со множеством лиц — согласие на запись в ответственности клиента
